== Description ==

Microsoft Calculator clone for website

You can add shortcode [wopo-calculator] to page you want to show app windows

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wopo-calculator` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add shortcode [wopo-calculator] to page you want to show app windows
